# 💰 ExpenseTracker

A modern, responsive web application for tracking and managing personal expenses efficiently. Built with cutting-edge technologies to provide a seamless user experience for financial management.

## 🎯 About

ExpenseTracker is a full-featured expense management application that helps users:
- **Track Expenses**: Log and categorize all your spending
- **Visualize Spending**: View detailed charts and analytics of your expenses
- **Manage Budgets**: Set budgets and monitor spending patterns
- **Organize Transactions**: Sort, filter, and search through expenses easily
- **Data Insights**: Get comprehensive reports on spending habits

## 🔧 Tech Stack

### Frontend
- **React 18.3** - Modern UI library for building interactive components
- **TypeScript** - Type-safe JavaScript for robust code
- **Vite** - Lightning-fast build tool and development server
- **React Router v6** - Client-side routing for navigation
- **React Hook Form** - Efficient form handling and validation
- **Zod** - Schema validation for type-safe data validation

### UI & Styling
- **Tailwind CSS** - Utility-first CSS framework for rapid UI development
- **Shadcn/ui** - High-quality, unstyled component library built on Radix UI
- **Radix UI** - Headless UI components (Accordion, Dialog, Select, Tabs, etc.)
- **Lucide React** - Beautiful, consistent icon library

### Data & State Management
- **TanStack React Query** - Powerful server state management and caching
- **React Hook Form + Zod** - Form validation and submission

### Additional Libraries
- **Recharts** - Responsive charting library for data visualization
- **date-fns** - Modern date utility library
- **Sonner** - Elegant toast notifications
- **next-themes** - Theme management (light/dark mode)
- **React Resizable Panels** - Flexible panel layouts
- **Embla Carousel** - Touch-friendly carousel component

### Development Tools
- **ESLint** - Code quality and style enforcement
- **TypeScript ESLint** - TypeScript-specific linting rules

## 🎨 Features

✨ **Intuitive Interface** - Clean, modern UI built with Shadcn and Tailwind CSS  
📊 **Data Visualization** - Beautiful charts powered by Recharts

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm, yarn, or bun

### Installation

\`\`\`bash
# Clone the repository
git clone https://github.com/KaranSingh2608/ExpenseTracker.git
cd ExpenseTracker

# Install dependencies
npm install
# or
yarn install
# or
bun install
\`\`\`

### Development

\`\`\`bash
# Start development server
npm run dev
# or
yarn dev
# or
bun dev
\`\`\`

The application will be available at \`http://localhost:5173\`

### Build

\`\`\`bash
# Build for production
npm run build
# or
yarn build
# or
bun build
\`\`\`

### Preview

\`\`\`bash
# Preview production build
npm run preview
# or
yarn preview
# or
bun preview
\`\`\`
